# Student_Placement_Analysis-Prediction
Student Placement Prediction

This project predicts whether a student will get placed based on academic and skill-related attributes using machine learning models.

Project Goal
Build a classification model to predict student placement outcomes using supervised learning algorithms.

Dataset Features
CGPA
Internships
Projects
Communication Skills
Logical Reasoning
Technical Skills

Machine Learning Models Used
Logistic Regression
Decision Tree
Random Forest
Support Vector Machine

Model Evaluation
Accuracy
Precision
Recall
F1 Score

Technologies Used
Python

NumPy
Scikit-learn
Matplotlib
Seaborn
